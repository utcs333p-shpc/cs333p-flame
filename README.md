# FLAMEPROF

This is a sample project for Assignment 5, which can be used as your own starter code. 

Directory: `flameatlab`
    Contains the MATLAB files needed to run the FLAME implementation of your
algorithm. 

Note: This sample project includes the derivation and implementation of both
unblocked and block variants. However, for this assignment you will be deriving
and implementing only the unblocked variants.
